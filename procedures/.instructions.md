# Procedure Page Rules

Use `procedures/_TEMPLATE.md` as the structural reference.

## Section Order

1. H1 title (`[Tool Name] — Walkthrough`) → Metadata line (Tool type · Source) → Intro paragraph → Contents → When to Use This Tool → Prerequisites → Step N sections → [Optional end sections] → Footer

## Step Format

- Each step is an H2: `## Step 1 — [Action]`
- Subsections within steps use H3
- Include `> [!TIP]` and `> [!NOTE]` where helpful
- Each step should be independently actionable

## Optional End Sections

Interpret the Results · Common Scenarios · Troubleshooting · Accuracy and Limitations · Customisation · Related Tools

## Footer

Always end with: `[← Back to Procedures](README.md) · [← Back to Sentinel Maturity Model](../README.md)`
